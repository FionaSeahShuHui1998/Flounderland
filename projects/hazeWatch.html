Introduction

HazeWatch is a simple web application that displays real-time PSI (Pollutant Standards Index) readings across Singapore using data from the Data.gov.sg PSI API.

Tech Stack

  • JavaScript
  • Data.gov.sg PSI API
  • CSS
  • HTML

Challenges Faced

Challenge 1: Data Accuracy & Consistency

Problem:

Ensuring PSI data across multiple regions and indicators was accurate and consistently displayed.

Solution:

Validated API responses and mapped each indicator carefully to ensure correct values were shown for every region.

Challenge 2: Presenting data-heavy information clearly

Problem:

Displaying multiple PSI metrics without overwhelming users required careful UI design.

Solution:

Organized the data into a structured table layout and adjusted spacing and labels to improve readability.

Challenge 3: Deployment and environment constraints

Problem:

Browser security restrictions affected API access when running the project locally.

Solution:

Deployed the application using GitHub Pages to ensure consistent behavior and easier access across environments.
