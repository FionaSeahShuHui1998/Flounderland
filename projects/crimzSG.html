Introduction

CrimzSG is a web-based data application that retrieves and analyzes crime statistics in Singapore from 2011 to 2024 using publicly available data from the Singapore Government’s official data platform.

Tech Stack

  • JavaScript
  • Data.gov.sg Crime Data API
  • CSS
  • HTML

Challenges Faced

Challenge 1: Working with large historical datasets

Problem:

Handling crime data spanning multiple years required managing a large dataset while ensuring the application remained responsive.

Solution:

Structured the data retrieval and processing logic efficiently to ensure smooth loading and clear presentation of long-range historical data.

Challenge 2: Data consistency across crime categories

Problem:

Different crime categories and yearly records needed to be consistently mapped and displayed for accurate analysis.

Solution:

Validated the API response structure and standardized how each crime category was processed and presented.

Challenge 3: Presenting data clearly for analysis

Problem:

Displaying multi-year crime statistics in a way that was easy to understand without overwhelming users required careful layout planning.

Solution:

Organised the data into a structured and readable format, focusing on clarity and logical grouping of information.
